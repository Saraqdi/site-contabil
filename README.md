# Contábil · Gestão — Site de Serviços Contábeis

Site institucional desenvolvido para divulgar serviços contábeis, com foco em comunicação direta com o cliente via WhatsApp e e-mail.

![Preview do site](https://via.placeholder.com/800x400?text=Site+Contábil+Gestão)

---

## 🚀 Sobre o projeto

Este é um site **one-page** responsivo para uma consultoria contábil. Ele apresenta:

- **8 serviços principais** (Folha de Pagamento, Abertura de Empresas, Dashboards BI, BPO Financeiro, MEI/Simples, Automação Contábil, Perícia Trabalhista e Auditoria/Compliance)
- **Seção de notícias e novidades** com atualização semanal
- **Links diretos para WhatsApp e e-mail**

---

## 🛠️ Tecnologias utilizadas

- **HTML5** — estrutura do site
- **CSS3** — estilização e responsividade
- **Font Awesome** — ícones vetoriais
- **Google Fonts** (via sistema) — tipografia
